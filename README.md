# 掀树杯#4 规则

本比赛采用明日方舟集成战略模式《岁的界园志异》作为比赛主题。

约定：禁止电表倒转；进入三层前有一次重开机会；取钱不得超过 50 块，超出部分按规则扣分。

最终得分 = 结算分 + 后勤分 + 作战分 + 限制分

## 结算分

结算分 = 原始得分 × 难度倍率

| 难度倍率 | |
| --- | ---: |
| N12 | ×0.8 |
| N13 | ×0.9 |
| N14 | ×1.0 |
| N15 | ×1.1 |
| N16 | ×1.2 |
| N17 | ×1.3 |
| N18 | ×1.5 |
